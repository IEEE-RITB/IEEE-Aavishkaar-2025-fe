## How to Setup?

*Steps to setup*

1. Clone the repository

```bash
git clone https://github.com/TheShiveshNetwork/IEEE-Aavishkaar-2025-fe
```

2. Go inside the clones directory and install the dependencies

```bash
cd IEEE-Aavishkaar-2025-fe
npm install
```

3. Run the server

```bash
npm run dev
```
